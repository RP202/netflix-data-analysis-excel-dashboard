Netflix Data Analysis Excel Dashboard 📊
## Project Overview
This project analyzes Netflix movies and TV shows using Microsoft Excel to identify trends in content type, genres, release years, and distribution of content on the platform.
The goal of this project was to explore the dataset and build an interactive dashboard to visualize key insights.

## Dataset
The dataset used in this project is from Kaggle:

Netflix Movies and TV Shows Dataset  
https://www.kaggle.com/datasets/shivamb/netflix-shows

## Key Insights
Netflix content significantly increased after 2016.
Movies make up a larger portion of the platform compared to TV shows.
Drama and Comedy are among the most frequent genres.
